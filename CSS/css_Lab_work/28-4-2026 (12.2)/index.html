Display : Flex (Parent Property) Justify-Content

The justify-content property aligns the flex container's items when the items do not use all available space on the main-axis (horizontally).

The justify-content property can also be used to align grid items in the inline direction. For pages in English, inline direction is left to right and block direction is downward.

Display : Flex (Parent Property) Justify-Content : flex-start;

"flex-start" in the context of CSS Flexbox refers to a value that determines the alignment of items within a flex container along the main axis, typically placing them at the start of the container.

Display : Flex (Parent Property) Justify-Content : flex-end;

In the context of CSS Flexbox, "flex-end" is a value used for the justify-content and align-items properties. It aligns items to the end of the flex container along the main or cross axis, respectively.

To center an element using Flexbox, set the parent container to display: flex, then use justify-content: center for horizontal alignment.

The items are evenly distributed within the alignment container along the main axis. The spacing between each pair of adjacent items is the same. The first item is flush with the start edge, and the last item is flush with the end edge.

Display : Flex (Parent Property) Align-items

The CSS align-items property sets the align-self value on all direct children as a group. In flexbox, it controls the alignment of items on the cross axis. In grid layout, it controls the alignment of items on the block axis within their grid areas.

Display : Flex (Parent Property) Align : flex-start;

The items are packed flush to each other toward the start edge of the alignment container in the appropriate axis.
